#API of the Helix Stocks Manager App
## Author: Jon Bonso

### How to Run Locally

1. Install the dependencies using this command: `npm install`
2. Start the API using: `npm start`
3. To verify, go to http://localhost:3000

### Deployed as well to Heroku
